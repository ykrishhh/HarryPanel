# Screenshots

To add screenshots:

1. Start HarryPanel locally: `cd harry-backend && python app.py`
2. Open `http://localhost:5000` in your browser
3. Take screenshots of:
   - Dashboard (main view)
   - Process Manager
   - File Manager
   - Database Admin
   - Terminal
4. Save them as PNG files in this directory
5. Reference them in the main README.md

## Recommended screenshot sizes

- Dashboard: 1200x800px (full-width hero)
- Feature screenshots: 800x600px
- Mobile views: 375x812px

## Tools for screenshots

- macOS: Cmd+Shift+4 (selection) or Cmd+Shift+5 (screencast)
- Windows: Snipping Tool or Win+Shift+S
- Linux: GNOME Screenshot or Flameshot
- Browser: Chrome DevTools Device Mode for mobile views

## GIF recording

For animated demos, use:
- [LICEcap](https://www.cockos.com/licecap/) (Windows/macOS)
- [Peek](https://github.com/phw/peek) (Linux)
- [Gifski](https://gifski.com/) (macOS)
- [asciinema](https://asciinema.org/) + [agg](https://github.com/asciinema/agg) for terminal demos
